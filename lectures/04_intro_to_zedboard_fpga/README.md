# Lecture 4. Introduction to ZedBoard FPGA

## Outline

1. Introduce the ZedBoard, FPGA fabric, and the board clock.
2. Identify switches, pushbuttons, LEDs, and reset inputs.
3. Explain the flow from Verilog to synthesis, bitstream, and programmed FPGA.
4. Introduce the top-level module and pin-constraint file.
5. Run a known-good LED or counter ckt before using matmul.
6. Connect `start`, `busy`, and `done` to simple board I/O.
