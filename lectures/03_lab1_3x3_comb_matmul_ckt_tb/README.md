# Lab 1: 3x3 Combinational Matmul Ckt and Testbench

## Outline

1. Refresh 3x3 matrix multiplication and calculate one output dot product by hand.
2. Map matrix entries to the flattened Verilog input vectors.
3. Complete the nested-loop combinational matmul ckt.
4. Read the testbench inputs, expected outputs, and pass/fail checks.
5. Run the ckt and testbench in EDA Playground.
6. Inspect a waveform and diagnose one intentional indexing or width error.
